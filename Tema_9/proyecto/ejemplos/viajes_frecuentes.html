HTML5 ha llegado para quedarse

HTML5 es la última versión de HTML. El termino representa dos conceptos diferentes: Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos. Contiene un conjunto más amplio de tecnologías que permiten a los sitios Web y a las aplicaciones ser mas diversas y de gran alcance. A este conjunto se le llama HTML5 y amigos, a menudo reducido a HTML5.

CSS3 la nueva versión de CSS

CSS significa Cascade Style Sheets, también llama Hojas de Estilo en Cascada. CSS es un lenguaje de marcado que se emplea para dar formato a un sitio web. Es decir, funciona en conjunto con arcibos HTML. Por esta razón para crear un Web debes saber tanto HTML como CSS.
